🔗 إعداد GitHub

أدخل بيانات المستودع مرة واحدة — تُحفظ في متصفحك ولا تُرسل لأي خادم.

أنشئ Token في GitHub → Settings → Developer Settings → Fine-grained tokens.
الصلاحية المطلوبة: Contents → Read and Write
🔒 يُحفظ الـ Token في localStorage فقط — لا يُرسل إلا إلى api.github.com عبر HTTPS.
